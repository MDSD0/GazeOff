Privacy Policy

GazeOff is a Windows application for eye breaks, blink and posture reminders, and healthier screen habits. This policy explains how the application and the GazeOff website handle information.

Summary

GazeOff is local-first. It does not require an account, sell personal information, include advertising trackers, or upload your break activity and recovery statistics to a GazeOff cloud service.

Application settings and activity history are stored on your Windows device. The website, font delivery, GitHub, and email contact involve third-party services as described below.

Information stored on your device

GazeOff stores the following information locally so its reminders and statistics work:

  • break intervals, durations, modes, schedules, and reminder preferences;
  • appearance, sound, startup, pause, and placement settings;
  • completed, skipped, and delayed break counts;
  • local screen-session durations, recovery statistics, streaks, and recent history; and
  • application state needed for tray controls and reminders.

This information is stored at %APPDATA%\com.gazeoff.app\gazeoff.json. GazeOff does not upload that file to the developer.

Smart Pause and Windows signals

GazeOff reads limited Windows signals locally to decide when a reminder should wait. Depending on the features you enable, this can include:

  • keyboard and pointer idle time;
  • whether the foreground application is fullscreen and its executable name;
  • Windows microphone and camera activity indicators;
  • display power state and working hours; and
  • your current wallpaper when wallpaper blur is selected.

GazeOff does not record audio or video, capture screenshots, read document contents, or upload these signals. Wallpaper processing happens locally for display inside the break interface.

Information you choose to provide

If you email the developer or submit a GitHub Issue, you may provide your email address, GitHub profile information, feedback, diagnostic details, screenshots, or logs. This information is used to reply, investigate bugs, understand feature requests, and improve GazeOff.

GitHub Issues are public unless you contact the developer privately by email. Do not include confidential or sensitive information in a public issue.

Website, network requests, and third parties

The website uses Vercel Web Analytics to measure page visits, traffic sources, device categories, and interactions such as Microsoft Store, review, support, and GitHub link clicks. This helps evaluate launch traffic and improve the website. GazeOff does not use this data to identify individual application users or connect website visits to local app activity.

The following third parties may process standard request information such as your IP address, browser or device details, requested URL, and request time under their own policies:

  • Vercel hosts the GazeOff website and provides privacy-focused website analytics.
  • Microsoft Store distributes the signed application and provides aggregate acquisition, usage, and rating information to the publisher.
  • Google Fonts provides font files used by the current application interface and website.
  • GitHub hosts source code, release downloads, and public issue forms.
  • Email providers process messages you choose to send to the developer.

Those services operate under their respective privacy policies. GazeOff does not control their independent logging, security, or retention practices.

Sharing and sale of information

GazeOff does not sell personal information. Information you provide may be handled by the service you use to send it, shared when required by law, or used when reasonably necessary to protect users, the application, or the developer from fraud, abuse, or security threats.

Retention and deletion

You control the application data stored on your device. To remove it, quit GazeOff and delete %APPDATA%\com.gazeoff.app\gazeoff.json. Deleting the portable executable alone does not automatically delete this settings and history file.

To request deletion of information sent by email, contact mr.imcommon@gmail.com. Public GitHub contributions are subject to GitHub’s controls and retention practices.

Children

GazeOff is a general-audience utility and is not designed to knowingly collect personal information from children.

Security

GazeOff limits application data to local storage and uses the operating system’s application-data location. No method of storage or transmission is completely secure, so absolute security cannot be guaranteed.

Changes to this policy

This policy may be updated when GazeOff’s features or services change. The latest version and its revision date will remain available at this URL.

Contact

For privacy questions or deletion requests, email mr.imcommon@gmail.com.
