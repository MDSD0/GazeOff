# Microsoft Store artwork

Upload these files manually in the English (United States) Store listing:

- `images/GazeOff-AppTile-300.png` to **1:1 App tile icon (300 × 300)**. This overrides the old icon with black bars.
- `images/GazeOff-BoxArt-1080.png` to **1:1 Box art (1080 × 1080)**.
- `images/GazeOff-Poster-1440x2160.png` to **9:16 Poster art (1440 × 2160)**.
- `images/GazeOff-SuperHero-1920x1080.png` to **16:9 Super hero art (1920 × 1080)**.

The super hero artwork intentionally contains no product title, as required by Partner Center. The clean transparent mascot source is `../microsoft-store-v0.1.3/images/GazeOff-Blob-Clean.png`.
